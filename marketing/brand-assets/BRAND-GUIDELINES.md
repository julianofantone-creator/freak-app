**Usage**: Code snippets, technical content, developer materials

- **Body**: 16px / 1rem (Regular) - Standard body text
- **Body Small**: 14px / 0.875rem (Regular) - Captions, metadata
- **Caption**: 12px / 0.75rem (Medium) - Labels, fine print

## üè∑Ô∏è Logo System

### Primary Logo
**Structure**: Wordmark "Freaky" with flame icon integration
**Colors**: Fire gradient (#FF4444 to #FF8844) on transparent background
**Usage**: Primary brand representation, headers, main CTAs

### Logo Variations

#### Horizontal Logo
- Full wordmark with icon
- Minimum width: 120px
- Usage: Website headers, business cards, email signatures

#### Stacked Logo  
- Icon above wordmark
- Square format optimized
- Usage: Social media profiles, app icons, square layouts

#### Icon Only
- Simplified flame symbol
- Usage: Favicons, social media, small applications
- Minimum size: 16px

#### Wordmark Only
- Text-only version without icon
- Usage: When icon isn't appropriate, tight spaces
- Maintains fire gradient treatment

### Logo Guidelines

#### Clear Space
- Minimum clear space: 1x the height of the logo on all sides
- No other elements should appear within clear space

#### Minimum Sizes
- Horizontal logo: 80px minimum width
- Stacked logo: 60px minimum width  
- Icon only: 16px minimum size
- Always maintain legibility

#### What NOT to Do
- ‚ùå Don't stretch or distort proportions
- ‚ùå Don't change colors outside brand palette
- ‚ùå Don't add drop shadows or effects  
- ‚ùå Don't use on busy backgrounds without proper contrast
- ‚ùå Don't rotate or skew the logo
